Betina Limpeza | Catálogo Digital

Uma solução leve e eficiente para pedidos online via WhatsApp, desenvolvida para facilitar a visualização e seleção de produtos de limpeza.

🚀 Funcionalidades

📱 Totalmente Responsivo

Layout adaptável para desktops, tablets e celulares. No mobile, o carrinho se transforma em uma "gaveta" (bottom sheet) moderna e acessível.

🔍 Busca Instantânea

Pesquisa em tempo real por nome, código ou marca, com debouncing para melhor performance.

🛒 Carrinho Dinâmico

Adicione, remova e ajuste quantidades facilmente. O estado do carrinho é mantido durante a navegação na página.

💬 Pedido via WhatsApp

Gera automaticamente uma mensagem formatada com o resumo do pedido e envia diretamente para o WhatsApp da loja.

📊 Exportação Excel

Permite baixar o pedido em formato .xlsx para controle administrativo ou backup do cliente.

🛠️ Estrutura do Projeto

  • index.html: Contém toda a estrutura, estilos (CSS) e lógica da aplicação (JavaScript).
  • produtos.js: Arquivo de dados que alimenta o catálogo. Contém a variável global allProducts.
  • imagens/: Pasta onde devem ser armazenadas as imagens dos produtos.

💻 Tecnologias Utilizadas

HTML5 & CSS3

Uso de variáveis CSS, Flexbox, Grid e Media Queries.

Vanilla JavaScript

Lógica pura sem dependência de frameworks pesados (React, Vue, etc).

SheetJS (XLSX)

Biblioteca leve para geração de planilhas Excel no navegador.

⚙️ Como Adicionar Produtos

Para adicionar novos produtos, edite o arquivo produtos.js seguindo o padrão JSON:

var allProducts = [
    {
        "nome": "NOME DO PRODUTO",
        "codigo": "CÓDIGO123",
        "imagem": "imagens/nomedearquivo.jpg"
    },
    // ... outros produtos
];
